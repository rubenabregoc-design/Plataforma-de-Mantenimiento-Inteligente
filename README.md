# 🛡️ MantechPro — Ecosistema Técnico Maestro V7.0

MantechPro es la infraestructura líder en Panamá para la gestión inteligente de activos y servicios técnicos. Diseñada bajo un modelo **Asset-Centric**, la plataforma adapta su interfaz y herramientas dinámicamente según el portafolio real del cliente.

## 🏗️ Arquitectura de Capas (Modelo V5)

| Capa | Descripción | Módulos Clave |
|---|---|---|
| **Capa 1: Core** | Siempre visible, crítica para la operación. | Chat Encriptado, Wallet (Escrow), Mantech ID, Bóveda de Garantías. |
| **Capa 2: Activo** | Específica por vertical (Salud, PH, Construcción). | Auditoría de Combustible, Reportes de Ingeniería, Tablero PH. |
| **Capa 3: Portafolio** | Vista consolidada del cliente. | Diagnóstico Predictivo, ROI Consolidado, Alertas Cruzadas. |

## 📦 Estructura de Planes Comerciales

*   **🔵 GRATIS ($0):** 2 activos, diagnóstico manual, 1 uso SOS/año.
*   **📦 EMPRENDEDOR ($29):** 5 activos, rastreo satelital, panel global.
*   **⚡ PROFESIONAL ($89):** 25 activos, auditoría combustible, IA predictiva, sello ingeniería.
*   **🏢 ENTERPRISE ($199):** Unidades ilimitadas, multi-edificio, API externa, seguro incidencias.

## 🛠️ Staffing por Proyecto
Línea de negocio independiente que permite contrataciones por lotes (N técnicos) para misiones puntuales de alto volumen, reutilizando la infraestructura de seguridad y pago de MantechPro.
